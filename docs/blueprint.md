# **App Name**: ShopAttend

## Core Features:

- Role Selection: Allow users to select their role: Admin or Employee.
- Admin Authentication: Secure login for admin users with email and password.
- QR Code Generation: Generate QR codes valid for a specified time window (start and end time).
- QR Code Scanner: Enable employees to scan QR codes to mark their attendance.
- Attendance Recording: Record employee attendance with timestamp upon successful QR code scan.
- Employee Management: Allow admins to add, edit, and deactivate employee accounts.
- Data Export: Provide admins with the ability to export attendance data in CSV or PDF format.

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A) for a professional and trustworthy feel.
- Background color: Light Gray (#F0F3F5), a very slightly desaturated version of the primary color.
- Accent color: Vibrant Purple (#A855F7) to highlight key actions and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral design.
- Use a set of consistent icons from a single library (e.g., FontAwesome or Material Icons) for navigation and actions.
- Employ a mobile-first, responsive design to ensure optimal viewing experience across devices. Bottom navigation bar is persistent across screens.
- Incorporate subtle transition animations for screen changes and feedback messages.
